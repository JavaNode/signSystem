## 🚀 启动方式

### 方式一：本地开发环境（推荐）
```bash
# 双击运行
start-local.bat
```

- 后端：http://localhost:8000

### 方式二：静态文件服务
```bash
# 双击运行  
serve-static.bat
```
- 使用构建好的dist文件
- 更接近生产环境

### 方式三：Docker部署（需要安装Docker Desktop）
```bash
# 安装Docker Desktop后运行
deploy.bat
```
